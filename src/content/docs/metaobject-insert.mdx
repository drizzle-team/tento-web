# Tento Metaobject Insert

Tento represents Shopify [Create Metaobject mutation](https://shopify.dev/docs/api/admin-graphql/2024-10/mutations/metaobjectCreate)

Tento provides you the most GraphQL-like way to insert rows into the Shopify.
<CodeTabs items={["index.ts", "schema.ts"]}>
<CodeTab>
```typescript copy
import { tento } from "@drizzle-team/tento";
import * as schema from "./schema";

const client = tento({ schema });

const result = await client.metaobjects.designer.insert({
  name: "Alex",
});
```
```typescript
// return type
const result: {
  _id: string;
  _handle: string;
  _displayName: string;
  _updatedAt: Date;
  name: string;
  description: string | null;
  website: string | null;
}
```
</CodeTab>
```typescript copy
import { metaobject } from "@drizzle-team/tento";

export const designer = metaobject({
  name: "Designer",
  type: "designer",
  fieldDefinitions: (f) => ({
    name: f.singleLineTextField({
      name: "Title",
      required: true,
      validations: (v) => [v.min(1), v.max(50)],
    }),
    description: f.multiLineTextField({
      name: "Description",
    }),
    website: f.url({
      name: "Website",
    }),
  }),
});
```
</CodeTabs>  

If you need insert type for a particular Metaobject Definition you can use typeof designer.$inferInsert syntax.
<CodeTab>
```typescript
type NewDesigner = typeof client.metaobjects.designer.$inferInsert;

const newDesigner: NewDesigner = { name: "Alex" };
const result = await client.metaobjects.designer.insert(designer);
```
