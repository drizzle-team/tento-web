### single_line_text_field
`string`  
A single-line text field. String value types can store up to five million characters.

```typescript
export const metaobjectDefinition = metaobject({
  name: "Metaobject Definition",
  type: "metaobject_definition",
  fieldDefinitions: (f) => ({
    field: f.singleLineTextField(),
  }),
});

/** A human-readable name for the field. This can be changed at any time. */
field: f.singleLineTextField({
  name: "text_field",
})

/** Whether metaobjects require a saved value for the field. */
field: f.singleLineTextField({
  required: true, // false as default
})

/** An administrative description of the field. */
field: f.singleLineTextField({
  description: "description",
})
```

<Callout collapsed="Validations">
Custom validations that apply to values assigned to the field.
* [Minimum length](https://shopify.dev/docs/apps/build/custom-data/metafields/definitions/list-of-validation-options#minimum-length)\
Sets the minimum length of a text value
```typescript
field: f.singleLineTextField({
  validations: (v) => [v.min(1)],
});
```
* [Maximum length](https://shopify.dev/docs/apps/build/custom-data/metafields/definitions/list-of-validation-options#maximum-length)\
Sets the maximum length of a text value
```typescript
field: f.singleLineTextField({
  validations: (v) => [v.max(1)],
});
```
* [Regular expression](https://shopify.dev/docs/apps/build/custom-data/metafields/definitions/list-of-validation-options#regular-expression)\
Sets a regular expression. Shopify supports [RE2](https://github.com/google/re2/wiki/Syntax).
```typescript
field: f.singleLineTextField({
  validations: (v) => [v.regex(".")],
});
```
* [Choices](https://shopify.dev/docs/apps/build/custom-data/metafields/definitions/list-of-validation-options#choices)\
A list of up to 128 predefined options that limit the values allowed for the metafield.
```typescript
field: f.singleLineTextField({
  validations: (v) => [v.choices(['choice1', 'choice2'])],
});
```
</Callout>
